# Aula sobre **_Condicionais_**

> #### Professor explicou sobre: IF Else, If aninhados, e resolveu uns exercicios conosco.

```C# 
if(condição){}
else{}
```
> ### If's aninhados

```C# 
if(condicao){}
else if(condição){}
else{}
```

> ### Explicou sobre operadores **And** ( && ) e **OR** ( | | )

```C#
A = 10, B = 10;

(A > 10 && B < 10); false

(A == 10 || B != 10); true
```
> ### Tabela Verdade **_And_**

> | **P** | **Q**| **P and Q**| 
> | :-: | :-: |  :-:  |         
> |  *True* | *False* |*False*| 
> |  *False* |*True* | *False*| 
> |  *False* |*False* |*False*| 
> |  *True*  | *True* | *True*| 

> ### Tabela Verdade **_Or_**

> | **P** | **Q**| **P Or Q**| 
> | :-: | :-: |  :-:  |         
> |  *True* | *False* |*True*| 
> |  *False* |*True* | *True*| 
> |  *True*  | *True* | *True*| 
> |  *False*  | *False* | *False*|
