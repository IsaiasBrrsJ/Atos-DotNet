# Laço de Repetição 

> #### for, while e do while
``` C#
for(; ;);
while(true);
do while(true);
```

## For

```C#
//para
for(int i = 0; i < 10; i++){}
```
```
Ele vai entrar em loop até que a condição seja satisfeita,
ou seja, foi atribuido o valor 0 para a várivel I,logo em seguida 
será feita uma verificação o valor que contém na váriavel I é menor que 10? (i < 10) 
se a condição imposta for verdadeira,
então entra na chave do FOR e executa
o bloco de codigo, depois incrementa +1 na variável I,
I passar a conter o valor de (0+1) ficando com 1.
sendo assim repete todo o bloco até que i não seja menor que 10.
```

## While
```C#
//enquanto
While(true){}
```
```
O while só vai entrar dentro do escopo das chaves caso, a condição seja satisfeita antes, e
se deixarmos TRUE dentro da condição,
ficará em um loop infinito,
para que isso não ocorra,
deveremos cololocar uma condição
de parada dentro do escopo da chave do laço while, ou definir uma 
condição no lugar do true.
```

## do While
```C#
//faça enquanto
do while(true){}
```
```
O do while, ao contrário do WHILE, ele primeiro entra no espoco
das chaves, para só depois testar a condição, ou seja, ele
sempre vai entrar por pelo menos uma vez no bloco de código.

```
